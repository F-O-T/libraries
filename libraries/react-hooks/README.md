# @f-o-t/react-hooks

A new FOT library

## Installation

```bash
# bun
bun add @f-o-t/react-hooks

# npm
npm install @f-o-t/react-hooks

# yarn
yarn add @f-o-t/react-hooks

# pnpm
pnpm add @f-o-t/react-hooks
```

## Quick Start

```typescript
import { hello } from "@f-o-t/react-hooks";

console.log(hello());
```

## API Reference

### `hello()`

Returns a greeting string.

```typescript
const greeting = hello();
console.log(greeting); // "Hello from @f-o-t/react-hooks!"
```
